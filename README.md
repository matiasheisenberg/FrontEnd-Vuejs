# Vue.js router
## proyecto de nuevas tecnologias PRUEBA

#### Para manejar sus dependencias
```
npm install
```

#### Compilar para desarrollo
```
npm run serve
```

#### Levantar servidor
```
cd src/
json-server --watch db.json
```

#### Compilar para produccion
```
npm run build
```
